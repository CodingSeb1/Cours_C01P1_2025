Description comportementale en SysML : le diagramme d'état (stm)

Le seul diagramme de description comportementale SysML au programme de PTSI est le diagramme d'état (stm). Ce diagramme est introduit ici sous forme d'exemple portant sur le système d'ouvre-portail FAAC. Il fera prochainement l'objet d'un cours à part entière portant sur les systèmes à événements discrets de type séquentiel.

Fondamental : Diagramme d'état (stm : state machine diagram)

Objectif : Décrire les différents états pris par le système pour un scénario d'utilisation donné.

Exemple : Diagramme d'état de l'ouvre-portail FAAC (scénario sans obstacle)
